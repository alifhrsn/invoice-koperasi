Rekening Pembayaran

Bila dikosongkan, invoice mencetak garis titik-titik untuk diisi tulis tangan. Data rekening tersimpan otomatis untuk invoice berikutnya.

Catatan & Cetak

• Ukuran kertas: A5 (atau custom 21 × 14,8 cm) · Orientasi: Landscape / Mendatar
• Margins: None / Tanpa margin · Scale: 100 (jangan "Fit to page")
• Buka More settings lalu centang Background graphics agar garis & warna tabel ikut tercetak.

        Langkah menyiapkan Supabase (sekali saja)
        
    
          
  1. Buka supabase.com → buat akun & proyek baru (gratis).
    2. 
          
  2. Di menu SQL Editor → New query → tempel SQL di bawah → Run.
    3. 
          
  3. Di menu Project Settings → API: salin Project URL dan anon key, tempel ke kolom di atas, lalu klik Hubungkan.
    4. 
          
  4. Ulangi langkah 3 di setiap perangkat lain (HP/komputer kasir) — data akan sinkron otomatis secara realtime.
    5.
